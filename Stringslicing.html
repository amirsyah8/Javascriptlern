String slicing dalam JavaScript adalah teknik untuk mengambil sebahagian daripada teks (string) berdasarkan indeks tertentu. Ini dilakukan dengan menggunakan kaedah slice() yang tersedia pada objek string. Dengan slicing, anda boleh mengekstrak bahagian string tanpa mengubah string asal. Sintaks slice():

string.slice(startIndex, endIndex);
